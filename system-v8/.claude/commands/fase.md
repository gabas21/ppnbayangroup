# /fase — Status Fase Saat Ini

## Peran Claude

Baca `PROJECT.md` bagian `PROGRESS SAAT INI` dan `ROADMAP`.

Tampilkan status dalam format ini:

```
📊 PROGRESS PROYEK: [Nama Proyek]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Fase saat ini : [nama fase]
Progress      : [estimasi %] selesai

✅ Sudah jalan     : [list]
🔄 Sedang dikerjakan: [item saat ini]
⏳ Berikutnya      : [fase/item selanjutnya]
🚧 Blocking        : [ada masalah? atau "Tidak ada"]

Timeline      : Mulai [tanggal] → Target [tanggal]
Sisa waktu    : [hitung dari tanggal hari ini]
```

Jika ada tanda ⚠️ KRITIS di fase berikutnya, ingatkan user untuk persiapan ekstra.
